# 🌦 WeatherX

A modern weather dashboard built using HTML, CSS, JavaScript, Fetch API, and OpenWeather API.

## 🚀 Features

* Real-Time Weather Data
* 5-Day Weather Forecast
* Current Location Support
* Dynamic Weather Icons
* Dynamic Backgrounds
* Dark / Light Theme
* Search History
* Mobile Responsive Design
* Error Handling
* Async/Await & Fetch API

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* OpenWeather API

## ⚙ Installation

1. Clone repository

```bash
git clone https://github.com/yourusername/weatherx.git
```

2. Open project folder

```bash
cd weatherx
```

3. Add your OpenWeather API key

```javascript
const API_KEY = "YOUR_API_KEY";
```

4. Run index.html
